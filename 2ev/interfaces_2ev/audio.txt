Formatos de audio
----------------- 
- sin comprimir {
	los mas precisos  
	los de mayor espacio
	EJEMPLOS:	- PCM (pulse code mod) usado para CD y DVD 
			- WAV (waveform audio file) windows
			- AIFF (audio interchange file) apple
		WAV y AIFF son contenedores PCM
  }
- compresion con perdida {
  	menor calidad
	menor tamaño
	EJEMPLOS: 	- MP3 (MPEG -1 Audio Layer 3)
			- AAC (Advanced Audio Coding) 
			      sucesor del MP3, tiene mejor calidad. usado para M4A
			- WMA (Windows media audio) exclusivo de Microsoft,
			      tiene mas calidad que MP3
			- OGG (Vorbis) contenedor audio+video+subtitulos
			- Dolby Digital usado para DVD, Blu-ray, streaming
  }
- compresion sin perdida {
	mayor tamaño
	EJEMPLOS: 	- FLAC (free lossless audio codec) 60%
			- ALAC (Apple free lossless audio codec)
			- WMA (Windows media audio) exclusivo de microsoft
			- Dolby True HD () home cinema
			- DTS-HD () Master audio
  }


RA: Real audio, se usa en programas como VC, solo se escucha para streaming, 
no se usa para ficheros. No tiene perdidas porque nos e almacena como tal
